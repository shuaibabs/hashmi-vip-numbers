# **App Name**: NumberFlow

## Core Features:

- Dashboard Summary: Display summary cards for key metrics like total numbers, RTS numbers, and pending uploads.
- Master Inventory Table: Display all numbers in a searchable and filterable data table with columns for mobile, status, purchase information, RTS date, location, and UPC status.
- RTS Status Update: Allow users to update the RTS status of a number via a modal, including adding notes and setting the RTS date.
- SIM Location Tracking: Track the location of SIM cards with a table displaying mobile number, current location, location type, and assigned user.
- Work Reminders: Manage work reminders with task name, assigned user, status, and due date. Allow marking tasks as done.
- Excel Import/Export: Import and export number data via Excel files (simulated with dummy data).
- Role-Based Access Control: Implement role-based access control to restrict access to certain pages and functionalities based on user roles (Admin/Employee).

## Style Guidelines:

- Primary color: Indigo (#667EEA) to give the app a professional and trustworthy look.
- Background color: Light gray (#F7FAFC), a subtle background that maintains focus on data.
- Accent color: Teal (#4FD1C5) for interactive elements to give clear visual cues.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, is suitable for both headlines and body text.
- Use lucide-react icons for a consistent and clean visual experience.
- Responsive layout with collapsible sidebar on mobile.
- Subtle animations for modals and status updates to enhance user experience.
